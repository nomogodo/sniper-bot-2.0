# 🔥 Solana Scalping Extremo Bot

Bot de trading automatizado para detectar y operar tokens nuevos en **Raydium** y **Pump.fun** con estrategia de scalping e interés compuesto.

## ⚠️ ADVERTENCIA

> Este bot es **SOLO PARA FINES EDUCATIVOS**. El trading de criptomonedas conlleva riesgos significativos. Nunca inviertas dinero que no puedas permitirte perder.

## 🚀 Instalación

```bash
# Clonar repositorio
git clone https://github.com/tu-usuario/solana-scalping-bot.git
cd solana-scalping-bot

# Instalar dependencias
npm install

# Configurar variables de entorno
cp .env.example .env
# Editar .env con tus credenciales

# Ejecutar
npm start
